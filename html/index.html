jQuery Plugin: zoomtabs

Description

The jQuery zoomtabs plugin is an enhancement of the plugin demonstrated by Remy Sharp on his blog jQuery For Designers.

Here, I have expanded on some of the functionality and also made a few minor tweaks that will allow a more streamlined implementation. The goal for this plugin is to take an set of images and display them in a menu that when hovered will zoom the image out and zoom the menu in. This effect mimics a flash implementation that was seen here.

Options

name description
zoomPercent The amount of zoom that is applied to the image.
speed The speed at which the menu is shown and the image is zoomed.
easing This plugin relies on the jquery.easing plugin. For this reason you may use any of the easing definitions that are defined there.
